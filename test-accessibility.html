Stairway Sprint - Accessibility Features Test

Implemented Accessibility Features:

  • Keyboard navigation for all menus (Tab, Shift+Tab, Arrow keys)
  • Visual focus indicators with yellow outline
  • Color-blind friendly modes (Protanopia, Deuteranopia, Tritanopia)
  • High contrast mode option
  • UI sound effects with adjustable volume
  • Screen reader support (ARIA live regions)
  • Proper contrast ratios (WCAG 2.1 compliant)
  • Settings persistence in localStorage

Keyboard Shortcuts:

Navigate menus Tab / ↑ ↓
Select/Activate Enter / Space
Pause game ESC / P
Move player ← → / A D

Testing Instructions:

  1. Keyboard Navigation: Press Tab to navigate through menu items. Yellow outline shows focus.
  2. Settings Access: Select "Settings" from main menu or pause menu.
  3. Color Blind Mode: In settings, choose different color blind modes to see visual adjustments.
  4. High Contrast: Toggle high contrast mode for improved visibility.
  5. Sound Volume: Adjust UI sound volume with + and - buttons.
  6. Screen Reader: Enable screen reader mode for ARIA announcements.
